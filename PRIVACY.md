# Privacy Policy

FileLinkKekUpload uploads files you choose to send with Thunderbird FileLink to the KekUpload server you configure.

The extension stores your configured KekUpload instance URL, upload chunk size, and file-name preference locally in Thunderbird using extension storage. If enabled, the original attachment name is sent to the configured server during upload. Attachment contents are always sent to the configured server when you use FileLink with this provider.

After an upload is finished, FileLinkKekUpload cannot delete that uploaded file from the configured server. If you need to remove an uploaded file, delete it directly from the server.

FileLinkKekUpload does not send data to any server other than the KekUpload instance you configure, does not include analytics, and does not sell or share your data.
